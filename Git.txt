1.  Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

В командной строке выполнить команду git clone, а затем вставить скопированный URL репозитория

2. Как получить на локальный компьютер изменения с github.com

С помощью git pull

3.git status - Эта команда показывает текущее состояние вашего локального репозитория.
git  log - Эта команда показывает историю коммитов в локальной ветке.
git remote -v - Эта команда показывает список удаленных репозиториев, связанных с вашим локальным репозиторием.
