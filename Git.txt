1.  Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

В командной строке выполнить команду git clone, а затем вставить скопированный URL репозитория

2. Как получить на локальный компьютер изменения с github.com

С помощью git pull

3.git status - Эта команда показывает текущее состояние вашего локального репозитория.
git  log - Эта команда показывает историю коммитов в локальной ветке.
git remote -v - Эта команда показывает список удаленных репозиториев, связанных с вашим локальным репозиторием.
=================================================================================================================
1) Чтобы создать Git репозиторий из локального каталога на своем компьютере и отправить его на GitHub:
понадобится команды как: 
1. git init
2. git add
3. git commit -m "Ваше сообщение коммита"
4. git remote add origin <URL репозитория>
5. git push -u origin <имя ветки>

2) Чтобы отказаться от изменений в файле(файлах), находящихся в рабочем каталоге (work directory), и вернуться к последнему коммиту, нужно использовать команду git checkout -- <имя файла>

3) Чтобы вернуть файл из индекса в рабочий каталог, нужно использовать команду git checkout -- <имя файла>

4) Чтобы отменить ошибочный коммит, нужно использовать команду git revert
